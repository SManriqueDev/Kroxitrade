# Settings Improvements

## Espanol

Desde la ultima release, Poe Trade Plus sumo mejoras importantes en usabilidad, compatibilidad y estabilidad. Se agrego soporte para mas regiones del sitio de trade de Path of Exile, incluyendo subdominios localizados y Garena, junto con una interfaz traducida a varios idiomas para ofrecer una experiencia mas comoda segun cada region. Tambien se incorporo la nueva pestana de Bulk Sellers para detectar vendedores repetidos en los resultados, nuevos iconos de navegacion, una organizacion visual mas clara del panel y ajustes en el ancho por defecto del sidebar para lograr una disposicion mas consistente.

En calidad de vida, ahora los nombres largos de las busquedas guardadas muestran tooltip, el selector de idioma fue reemplazado por un dropdown localizado, y el estado de la interfaz se guarda por version del trade para evitar mezclar configuraciones entre entornos. Ademas, se mejoro el sistema de precios equivalentes con una actualizacion mas confiable y una cache mas corta para reflejar mejor los valores de poe.ninja. Entre los bugs resueltos, se corrigieron problemas de refresco en los equivalentes de precio, errores relacionados con contextos invalidos de la extension, inconsistencias en el historial y el estado persistido, y varios detalles de accesibilidad en carpetas y navegacion.

## English

Since the last release, Poe Trade Plus has received major improvements in usability, compatibility, and stability. Support was added for more Path of Exile trade site regions, including localized subdomains and Garena, along with a translated interface in multiple languages to provide a smoother experience for each region. A new Bulk Sellers tab was also introduced to detect repeated sellers in search results, together with new navigation icons, a cleaner panel layout, and adjustments to the default sidebar width for a more consistent presentation.

Quality-of-life updates include tooltips for long saved search names, a localized language dropdown replacing the previous selector, and version-aware UI state persistence so settings do not get mixed between trade environments. Equivalent price handling was also improved with more reliable refresh behavior and a shorter cache duration to better reflect poe.ninja values. Fixed bugs include stale equivalent price updates, errors caused by invalid extension contexts, inconsistencies in history and persisted state, and several accessibility issues across folders and navigation.
